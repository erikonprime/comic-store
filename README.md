**Commit prefix:**

1. One of the following:

    1. ```feat - changed or newly developed feature```
    2. ```fix - bug fix```
    3. ```chore - update documentation, update dependencies, update framework, etc.```
    4. ```refactor - refactored code```
    5. ```style - only styling changes```
